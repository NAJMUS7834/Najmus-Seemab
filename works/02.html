KYO

Know Your Officials

Production period

2019-2020 about 18 months

In charge of production

Planning/Analyzing/Designing/Developing

Skills used

Android / Java / XML / GoogleMap / FireStore / API's

Concept

This application makes appointments convenient and easy it has enabled features like AutoGps Update, which automatically update user availability status based on his device location.

Idea

Many time I faced issues while taking appointment with somone so I decided to make Appointment Easy.

Survey / Prototype

After disscussing with my friends and convencing them to collabrate with me we came up with the prototype and disscuss prototype with the potential user(shop owner, teacher, officials, students) of the application and make some modification in prototype.

Develop

Based on prototype we listed out all the features and started developing each features and after well testing added it to main app.
