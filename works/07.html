AOES

Advance Online Examination System

Production period

2020 about 2 month

In charge of production

Planning / Designing / Developing

Skills used

HTML / CSS / JS / PHP / SQL / Bootstrap

Concept

Complete Advance and Secure Test Manage System with both admin and Student Panel for conducting online quiz and test where detail score materics will be displayed to user and admin.

Initial Idea

As a co-founder of Anveshan club VVCE which help students to learn new technical skills. I with my other peers decided to develop an Online Examination System.

Starting

Initially we started with only MCQ. where Admin add some question and there multiple option and user can attend test and see there score.

Journey to Advance

After meeting basic requirement we upgraded our system by giving freedom to admin (adding any type of question) and user (see there performance matrix, percentile, group rank, question explanation if added by admin).

Achivements

Praised by HOD and college in-house start-up took the project.
10 unit of project sold till today to various organization
